# Academic paper example

A minimal professor-facing Typst paper generated from TypstRAG source evidence.

Compile:

```bash
typst compile examples/academic-paper/main.typ examples/academic-paper/main.pdf
```

It demonstrates:

- title and author block
- abstract
- two-column body
- figure caption and reference
- bibliography citation and bibliography list

Source evidence: [`docs/RAG-SOURCES.md`](../../docs/RAG-SOURCES.md).
